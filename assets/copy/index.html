Welcome to a collection of various tools to do various things relating to the game Astrox Imperium. Well it's not really a collection at the moment as there is only one tool (displayed below), but more should appear over time and be accessible via the side navigation. For the security conscious, rest assured everything is processed via client-side only JavaScript - the data you enter here is not sent to any servers. Google Analytics and Tag Manager are only used to help me understand how people use this site - it's easy to block if you wish.

Ship Info Exporter

Provided with the contents of a ship data file, this tool will allow you to customise and export the data into useful formats such as BBCode, Markdown (eventually) and HTML (also eventually). The output can then be pasted into other websites, for example Steam discussions or Nexus Mods.


description Source Ship Data

info Ship data files can be found in the Astrox Imperium_Data\MOD\ships folder, inside your Astrox Imperium install directory. File input is a planned feature!

web Template Editor

warning Note: this will remove any changes you have made in the template editor!

Click a label to insert its Name and Value tags into the template at the cursor position. Shift+click inserts just its Name tag. Alt+click inserts just its Value tag.

Due to bugs with the editor, label insertion has been disabled while it is in BBCode mode. Switch back to normal mode to be able to insert labels again.

Once you're happy with the template, press Generate Output to generate code that can be copied to other websites.

info Saving templates, "Share" features and more code formats (e.g. Markdown, HTML) are all planned to be added.
